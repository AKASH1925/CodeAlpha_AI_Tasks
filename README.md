# CodeAlpha AI Tasks

This repository contains AI-based projects completed as part of my **CodeAlpha Internship**.  
The projects demonstrate applications of Artificial Intelligence, Natural Language Processing (NLP), Deep Learning, Computer Vision, and AI-powered user interfaces.

## Projects Included

## Task 1: Language Translation Tool 

### Description
A language translation application that allows users to enter text, select source and target languages, and receive translated output using a translation API.

### Features
- User-friendly interface for entering text
- Source and target language selection
- Translation using API services
- Displays translated text clearly
- Copy translated text feature
- Text-to-speech support (optional)

### Technologies Used
- React / TypeScript
- Translation API (Google Translate API / Microsoft Translator)
- HTML, CSS, JavaScript

# Task 3: AI Music Generation

### Features

- AI model training
- Music pattern learning
- Generation of new music sequences

### Technologies Used

- Music21
- TensorFlow / PyTorch
- RNN / LSTM / GAN models
- Deep Learning
