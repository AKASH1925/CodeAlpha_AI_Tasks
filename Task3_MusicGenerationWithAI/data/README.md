# MIDI Dataset
## Getting MIDI Files
### Recommended Sources (Free)
1. **Kaggle MIDI Dataset**
   - https://www.kaggle.com/datasets/simsocl/lakh-midi-dataset
   - Large collection of classical music
2. **MuseScore**
   - https://musescore.com/sheetmusic
   - Free sheet music that can be exported as MIDI
3. **IMSLP**
   - https://imslp.org/
   - Classical scores (MIDI versions available)
4. **Classical MIDI Archives**
   - https://www.classicalarchives.com/
   - MIDI files for classical compositions
### Recommended Composers for Training
- **Mozart**: Simple, clear melodies
- **Bach**: Complex harmonies and counterpoint
- **Chopin**: Emotional, melodic pieces
- **Debussy**: Impressionistic patterns
### Setup Instructions
1. Download MIDI files from the sources above
2. Create a folder: `data/midi/`
3. Place all `.mid` files in that folder
4. Recommended: Start with 20-30 MIDI files for initial testing
